# Política de Privacidade — Blackboard+ PDF Downloader

Última atualização: 23 de março de 2026

## Coleta de dados
A extensão Blackboard+ PDF Downloader não coleta, armazena, transmite ou compartilha nenhum dado pessoal do usuário.

## Funcionamento
A extensão funciona exclusivamente no navegador do usuário. Ela acessa a API REST do Blackboard Learn para listar os arquivos disponíveis nos cursos em que o usuário está matriculado. Todas as requisições são feitas diretamente do navegador do usuário para o servidor do Blackboard da sua instituição. Nenhum dado passa por servidores intermediários.

## Permissões

- downloads: utilizada para salvar os arquivos selecionados no dispositivo do usuário.
- host_permissions (*.blackboard.com): utilizada para acessar a API do Blackboard Learn e listar os arquivos dos cursos.

## Armazenamento
A extensão não armazena nenhum dado localmente ou remotamente. Nenhum cookie, token ou informação pessoal é salvo pela extensão.

## Terceiros
Nenhum dado é compartilhado, vendido ou transferido a terceiros.
